## Как использовать файлы

### Для обновления событий:

1. **Подготовить CSV файл**
   ```csv
   title,zone,date,startTime,endTime,description
   "Открытие форума","main-hall","2026-02-15","09:00","10:00","Торжественное открытие"
   ```

2. **Конвертировать в JSON**
   ```bash
   node convert_events_csv.js events.csv
   ```

3. **Импортировать в приложение**
   ```bash
   node import_events.js events.json
   ```

4. **Пересобрать и задеплоить**
   ```bash
   npm run build
   firebase deploy
   ```

### Для обновления 3D модели:

1. **Экспортировать из Blender**
   - File → Export → glTF 2.0 (.glb)
   - Формат: GLB (binary)
   - Include: Selected Objects
   - Transform: +Y Up

2. **Заменить файл**
   ```bash
   copy new_model.glb public/SM_MFF.glb
   ```

3. **Пересобрать**
   ```bash
   npm run build
   firebase deploy
   ```

### Для обновления зон:

1. **Редактировать файл**
   ```typescript
   // src/data/mockData.ts
   export const zones: Zone[] = [
     {
       id: 'zone-1',
       name: 'Конференц-зал А',
       description: 'Главный зал',
       type: 'conference',
       position: [10, 0, 5],
       color: '#4A90E2'
     },
     // ...
   ]
   ```

2. **Пересобрать**
   ```bash
   npm run build
   firebase deploy
   ```

## Требования для самостоятельной работы

### Необходимое ПО:
1. **Node.js** (v18 или выше)
   - Скачать: https://nodejs.org/

2. **Git** (для работы с репозиторием)
   - Скачать: https://git-scm.com/

3. **Firebase CLI** (для деплоя)
   ```bash
   npm install -g firebase-tools
   ```

4. **Blender** (для работы с 3D моделью)
   - Скачать: https://www.blender.org/

5. **VS Code** (рекомендуемый редактор)
   - Скачать: https://code.visualstudio.com/

### Установка проекта:

```bash
# 1. Клонировать репозиторий
git clone https://github.com/akmaltong/forum-nav-app.git
cd forum-nav-app

# 2. Установить зависимости
npm install

# 3. Запустить локально
npm run dev

# 4. Открыть в браузере
# http://localhost:3000
```
