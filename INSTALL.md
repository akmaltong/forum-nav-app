# Быстрая установка

## Шаг 1: Установка зависимостей

Откройте PowerShell или командную строку в папке `forum-nav-app` и выполните:

```powershell
npm install
```

Это установит все необходимые библиотеки (~5 минут).

## Шаг 2: Добавление 3D модели

Ваша 3D модель уже скопирована в папку проекта как `venue-model.glb`.

Если модель не загружается, приложение автоматически использует упрощённую версию.

## Шаг 3: Запуск приложения

```powershell
npm run dev
```

После запуска откройте браузер и перейдите по адресу:
**http://localhost:3000**

## Шаг 4: Тестирование на мобильном

Чтобы открыть на телефоне в той же сети:

1. Узнайте IP вашего компьютера:
```powershell
ipconfig
```

2. Найдите IPv4 адрес (например, 192.168.1.100)

3. На телефоне откройте: `http://192.168.1.100:3000`

## Возможные проблемы

### Ошибка "npm not found"
Установите Node.js с официального сайта: https://nodejs.org/

### Ошибка портов
Если порт 3000 занят, Vite автоматически выберет другой (3001, 3002 и т.д.)

### 3D модель не загружается
Убедитесь что файл `venue-model.glb` находится в папке `public/`

### Медленная загрузка
3D модель большая (258 МБ). Для продакшена рекомендуется оптимизация.
